Tipos de herramientas

Herramienta

Estas herramientas nos sirven para la jardineria dado que su uso es muy util, estas herramientas son una pala y diferentes tipos de rastrillos el valor de cada una es de alrededor de %15.000 a $30.000 pesos.

Manguera

Manguera

Esta herramienta se utiliza para regar las plantas y jardines, consta de diferentes tipos de riego con el cual se adhiere a como quiere regar cada persona. Su valor es de $10Mil Pesos



Tijera de Podar

Tijeras

Esta herramienta tiene como utilidad cortar u podar plantas y arboles, su valor es de %10.000 mil pesos.

Rastrillo

Rastrillo

Esta herramienta se utiliza para barrer diferentes zonas de jardines, pueden ser hojas de arboles, hojas de plantas y otras. Su valor es de $15Mil pesos.




Carretilla Jardin

Carretilla

Esta herramienta tiene diferentes tipos de uso, uno de ellos es recolectar tierra , el otro uso que puede tener es recolectar las hojas u ramas. Su valor es de $50Mil pesos.







Pala Jardinera

Pala Jardinera

Esta herramienta se utiliza para hacer hoyos con los cuales se pueden plantas arboles u diferentes tipos de plantas. Su valor es de $7Mil pesos.
